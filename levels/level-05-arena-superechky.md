# Рівень 5: Арена суперечки

## Загальна інформація

| Параметр | Значення |
|----------|----------|
| **Номер рівня** | 5 |
| **Назва** | Арена суперечки |
| **Навичка** | Визначати суть спору |
| **Тривалість** | 10–15 хвилин |

---

## Сюжет

### Локація
Кругла арена з трибунами. Посередині — два персонажі, які кричать один на одного. На підвищенні сидить суддя.

### Персонажі
- **Персонаж А** — гарячий, розмахує руками
- **Персонаж Б** — теж збуджений, перебиває
- **Суддя** — спокійний, намагається зрозуміти суть

### Діалог-вступ
```
Суддя: "Тихо! ТИХО!

        Ви кричите вже годину,
        але я досі не розумію —
        про що ви взагалі сперечаєтесь?

        Мандрівнику, допоможи!
        Послухай їхній спір
        і скажи — про що він НАСПРАВДІ?"
```

## Ігровий процес

### Етап 1: Аналіз діалогів

**Інструкція для гравця:**
«Прочитай діалог і визнач головну тему спору»

**Приклади діалогів:**

---

**Діалог 1:**
```
А: "Ти завжди береш мій олівець без дозволу!"
Б: "Та я ж поверну! Чого ти такий жадібний?"
А: "Я не жадібний! Просто попроси спочатку!"
Б: "Та це ж просто олівець!"
```

**Питання:** «Про що цей спір насправді?»

**Варіанти:**
| Варіант | Відповідь | Правильно? |
|---------|-----------|------------|
| A | Про олівець | ❌ |
| B | Про повагу і дозвіл | ✅ |
| C | Про жадібність | ❌ |

**Правильна відповідь:** B — спір про те, що потрібно питати дозволу

### Етап 2: Завершення рівня

```
Суддя: "Чудово!

        Тепер я розумію головне:
        Коли люди сперечаються —
        вони часто говорять про одне,
        а ДУМАЮТЬ про інше.

        Щоб вирішити спір —
        треба зрозуміти його СПРАВЖНЮ тему!"
```

## Домашнє завдання

### Формулювання
```
Суддя: "Домашнє завдання:

        Згадай один спір удома або в школі.

        Напиши:
        1. Про що говорили вголос?
        2. Про що спір був НАСПРАВДІ?

        Наприклад:
        Говорили про іграшку,
        а насправді — про справедливість."
```

### Поля для відповіді
- Текстове поле «Про що говорили вголос:»
- Текстове поле «Про що був спір насправді:»
- Кнопка «Зберегти»

## Підказки для розробників

### Візуальні елементи
- Арена з трибунами
- Два персонажі з «хмаринками» діалогу
- Суддя на підвищенні
- Анімація суперечки (руки, міміка)
- При правильній відповіді — персонажі заспокоюються

### Звукове оформлення
- Гул натовпу на фоні
- Голоси персонажів (можна текст)
- Спокійний голос судді
- Звук «розуміння» при правильній відповіді
